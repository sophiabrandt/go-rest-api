# go-rest-api

go-rest-api is a playground for me to explore how to write Go micro-services from scratch.

## Installation

The project uses Go modules, so you'll need Go >= 1.11.

```bash
git clone https://github.com/sophiabrandt/go-rest-api.git
```

## Usage

```bash
go run ./cmd/server
```
